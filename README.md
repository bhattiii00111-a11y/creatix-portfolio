# Creatix - Portfolio Website

## Next.js Portfolio Site

Built with Next.js 16, Tailwind CSS 4, TypeScript, shadcn/ui components.

## Requirements

- Node.js 20+ 
- npm or pnpm

## Setup & Run

1. Install dependencies:
   ```
   npm install
   ```
   or
   ```
   pnpm install
   ```

2. Run development server:
   ```
   npm run dev
   ```
   Open http://localhost:3000

## Build for production
```
npm run build
npm start
```

## Pages
- Home: /
- About: /about
- Projects: /projects
- Services: /services
- Reviews: /reviews
- Contact: /contact

## Deployment
Deploy to Vercel with one-click from GitHub repo.
